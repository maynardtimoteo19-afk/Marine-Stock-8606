☁️ Cloud Setup Guide
Step 1
Create a Google Account
Go to accounts.google.com/signup and create a free Google account if you don't have one.
Step 2
Create a Firebase Project
1. Go to console.firebase.google.com
2. Click "Add project" → Name it marinestock
3. Disable Google Analytics (not needed) → Click "Create project"
Step 3
Add a Web App & Get Config
1. In your project, click the </> (Web) icon
2. Register app name: marinestock-app → Click "Register app"
3. You'll see a firebaseConfig block — copy the whole JSON object
Step 4
Set Up Firestore Database
1. In Firebase sidebar: Build → Firestore Database
2. Click "Create database"
3. Choose "Start in test mode" → Select any location → Click "Enable"
Step 5
Paste Your Config Below
Paste only the values inside the curly braces { ... }
